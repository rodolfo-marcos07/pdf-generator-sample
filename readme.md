# Installation

1. clone the git repository `git clone git@github.com:rkristelijn/pdf-generator-sample.git`
2. go into the directory and install dependancies: `cd pdf-generator-sample || npm i`
3. `mkdir pdf`

# Running the sample

1. `node pdf-generator.js`
